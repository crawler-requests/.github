# `.github`

Shared config + **organization profile** for [**crawler-requests**](https://github.com/crawler-requests).

## Organization profile README

The profile shown on the [org homepage](https://github.com/crawler-requests) lives at **[`profile/README.md`](profile/README.md)** — a navigation hub across all `requests-<lang>` repositories.

Regenerate it from the live repo list:

```bash
bash gen-org-readme.sh profile/README.md
git commit -am "docs: refresh org profile" && git push
```
